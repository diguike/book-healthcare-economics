## 待核 / 口径提示

- **印度 20%/8%/57% 三数**：来源均为印度产业推广机构（InvestIndia/IBEF），量级在多份资料间一致，但属推广口径，正文已显式标注，未取到中性第三方（如 IQVIA）逐项复核值。
- **Humira 2022 峰值 212 亿**：归至 AbbVie FY2022 全年业绩公告（2023-02），本轮未单独 WebFetch 原始 release 行项，作为背景量级使用；2023（144.04 亿）、2024（89.93 亿）均已核到 AbbVie 公告原文。
- **Humira 约 247 项专利申请**：I-MAK 报告口径（产品上市后约九成递交）；I-MAK 为专利改革倡导机构，立场需知悉，但其专利计数被多家学术/媒体广泛引用。
- **中国原料药出口 2023（量 +5.4% / 额 -20.6% / 约 409 亿美元）、关停约 145 家**：转述自 C&EN/APIFDF/ECHEMI，原始口径为中国医保商会，本轮未回溯医保商会原表。
- **生物类似药单品开发成本 1–3 亿美元**：行业区间估计，非单一一手源，正文与 CSV 均标「估计」。
- **印度对华中间体依赖「六七成」**：业界估计，口径分歧大，正文已软化为「某些品类」。
- Hyrimoz 周处方 640→8300：机构追踪数据（BioSpace 援引），非 CVS/Sandoz 官方披露。
